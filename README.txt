Aplicación: Conversor Voz a Texto + Resumen con IA

Archivos generados:
- index.html: Interfaz principal
- style.css: Estilos visuales
- script.js: Lógica de grabación y resumen
- README.txt: Información del proyecto

Requisitos:
- Navegador compatible con Web Speech API (Chrome recomendado)
- Clave de API válida para Gemini (reemplazar en index.html)

Para ejecutar:
1. Abre Visual Studio Code en esta carpeta
2. Usa Live Server o ejecuta: python -m http.server 8000
3. Abre en navegador: http://localhost:8000
